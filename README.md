# Apartment Sale Management System

This is a C# Windows Forms project for managing apartment sales.

# Features

- Admin dashboard
- Customer registration and login
- Apartment management
- Customer list management
- Payment history management

# Technologies Used

- C#
- Windows Forms
- .NET Framework
- Visual Studio 2022

# Project Description

The Apartment Sale Management System is designed to manage apartment-related information, customer details, and payment records. It helps admin users manage apartment data and customer information through a simple Windows Forms interface.

# How to Run

1. Open the project in Visual Studio 2022.
2. Open `ApartmentSaleManagement.sln`.
3. Build the solution.
4. Run the project.
